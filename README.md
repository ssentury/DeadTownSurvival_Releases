# Dead Town Survival

The official release repository for the game Dead Town Survival.
This repository provides downloadable builds for supported platforms.

---

## Standard Version

The **Standard Version** is the latest stable release of the game, intended for general players.

### Android
- The game is primarily released through the **Google Play Store**.
- If you are unable to access the Play Store, you can download the Android build from this repository.

### PC
- The PC version is officially released through this repository.
- Download the latest build from the **Releases** section.

---

## Tester Version

The **Tester Version** download is available **only during public testing periods**.  
This version may include experimental features and is subject to change.

### Android / PC
- Tester builds for both Android and PC are released through this repository during test periods.
- Please refer to the **Releases** section for availability.

---

## Downloads
All downloadable files are provided via **GitHub Releases**.  
Go to the **Releases** tab to find the latest available versions.

---

## Notes
- Tester versions may be unstable and are not recommended for regular gameplay.
- Save file does not linked between tester and standard versions.
